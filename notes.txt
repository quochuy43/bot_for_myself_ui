Tóm gọn cách các file phối hợp:

App.jsx: quản lý state + logic gọi API.
ChatForm.jsx: nhập tin nhắn user, gửi request.
ChatMessage.jsx: hiển thị tin nhắn trong danh sách.
ChatbotIcon.jsx: biểu tượng con robot tái sử dụng.

Chi tiết
2 thằng đều Nhận props từ App.jsx
ChatMessage.jsx: Để cập nhật giao diện chatbot thôi chứ k có chi deep
ChatForm.jsx: Nhận input từ user => Thêm tin nhắn mới vào UI + Hiển thị "Thinking...", gọi generateBotResponse để lấy câu trả lời thật
